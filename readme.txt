== Description ==
**Paid Downloads Savano Gateway**

افزونه پرداخت آنلاین ساوانو این امکان فراهم می سازد تا به راحتی بتوانید برای پرداخت های افزونه دانلود به ازای پرداخت، از آن استفاده کنید

.

= امکانات =
* قابلیت نمایش خطاه های درگاه پرداخت

* سازگار با آخرین ورژن ووکامرس 

* رابط کاربری و تنظیمات ساده و آسان

* تنظیمات ارسال اطلاعات خریدار بصورت اختیاری

== Installation ==

1. برای نصب افزونه ابتدا در محیط مدیریت وارد بخش افزونه ها > افزودن > جستوجوی افزونه بروید در آنجا اسم افزونه را بنویسید.
2. با پیدا شدن افزونه بر روی گزینه هم اکنون نصب کن کلیک کنید و منتظر باشید تا افزونه نصب شود.
3. در این مرحله با کلیک بر روی دکمه فعال کردن افزونه خود را فعال کنید.
4. بعد از نصب افزونه از منوی مدیریت بر روی درگاه پرداخت و سپس تنظیمات کلیک کنید.
5. انتهای صفحه باز شده در فیلد کلید API، پین دریافتی از ساوانو را وارد کنید.
